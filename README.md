# Charge it
  Сейчас появляется все больше электроавтомобилей на дорогах России, но не у каждого автомобилиста есть собственный дом или место на паркинге с зарядной станцией. Таким автомобилистам приходится заряжать свой авто на ближайшей, найденной в интернете, электрозаправке, и есть большая вероятность того, что эта станция уже будет занята, и человек просто останется без авто на несколько часов, ведь заряжаются они достаточно долго.
  Именно для таких автомобилистов и создано приложение Charge it, с помощью которого любой обладатель электромобиля сможет найти ближайшую свободную электрозаправку и подзарядить свой авто.

Основные возможности:

- Отображение заправочных станций на карте
- Отображение текущего местоположения пользователя на карте
- Поиск объектов по запросу на карте
- Фильтрация заправочных станций по типу разъёма
- Отображение информации о заправочных станциях (адрес, расстояние до пользователя, типы и количество имеющихся разъёмов для зарядки и т. п.)
- Бронирование заправочных станций
  
Используемые технологии:
  
- Kotlin
- MVVM
- App modularization
- Coroutines
- Android Material design
- Android Jetpack Navigation
- SplashScreen API
- RxJava3
- Koin
- Room
- Google Maps API
